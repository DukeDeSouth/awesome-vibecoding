# Contributing

## Adding a tool

1. Fork the repo
2. Add to the appropriate table in `README.md`
3. PR title: `Add: ToolName`

## Adding a shipped project

Built something with vibecoding? Add it to the **Shipped Projects** table:

```markdown
| [Name](url) | What it does | Built with | Time to build | [GitHub](repo-url) |
```

## Adding a workflow or prompt

Add to the **Workflows** or **Prompts & Techniques** sections. Must be something you actually use, not theoretical.

## Criteria

- Tools: 100+ GitHub stars, actively maintained
- Projects: Must be live/shipped, with source code available
- Workflows: Must be battle-tested, not just ideas
